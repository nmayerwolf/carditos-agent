# Carditos Agent

Agente conversacional de rugby para el Club San Andrés. Ayuda a entrenadores de infantiles y juveniles con modalidades de juego, reglamento, manejo de chicos y ejercicios — via WhatsApp.

## Stack

Node.js 20 · TypeScript · Express · Supabase (Postgres + pgvector) · Anthropic Claude · OpenAI Embeddings · Kapso (WhatsApp) · Railway

## Setup

```bash
cp .env.example .env
# Completar variables en .env
npm install
npm run dev
```
